#### 개선 항목
- 한글 IME 입력 안정화: 조합 중(`isComposing`)에는 `Enter` 전송이 동작하지 않도록 수정.
- 키보드 동작 가이드 추가: 입력창 하단에 `Enter 전송 · Shift+Enter 줄바꿈` 힌트 표시.
- SQL 전송 전 안내 강화: SQL 모드 + DB 미선택 상태에서 전송 버튼 비활성화 및 경고 문구 표시.
- 첨부 파일 타입 확장: 이미지(`.jpg/.jpeg/.png/.gif/.webp/.bmp`)를 파일 선택창에서 직접 첨부 가능하게 변경.
- 첨부 파일 중복 제거: 이름/크기/수정시각 기준으로 중복 파일 자동 제외.
- 첨부 파일 개수 제한: 최대 10개로 제한하고 초과 시 경고 토스트 표시.
- 첨부 파일 관리 개선: 첨부 목록에 `전체 삭제` 버튼 추가.
- 첨부 버튼 상태 개선: 전송/분석 중 또는 제한 개수 도달 시 첨부 버튼 비활성화 스타일 적용.
